id : som2som3@dongduk.ac.kr

*실행 주의사항*
1. 회원가입 : 폼에 내용을 입력할 시에 이메일과 전화번호를 주어진 형식에 맞게 작성해주세요. 그렇지 않으면 알림창이 뜹니다. 
로그인하지 않은 상태에서 다른 페이지로의 이동을 누르면 로그인페이지가 뜹니다.

*테스트 루틴*
1. 로그인
이메일과 비밀번호를 누르고 로그인을 해주세요. 이 때 이메일이나 비밀번호를 입력하지 않고 로그인 버튼을 누르면 알림창이 뜨고 로그인이 진행되지 않습니다.
만약 회원가입 하지 않은 정보로 로그인하면 경고창없이 다시 로그인 창이 뜹니다. 

2. 마이페이지 조회 
로그인을 성공하면 마이페이지로 이동됩니다. 여기서 회원가입 할 때 입력했던 나의 정보들이 나옵니다. 

3. 회원정보 수정
마이페이지에서 하단의 수정하기를 누르면 성별과 나이대를 제외한 나머지 정보들을 수정할 수 있습니다. 
수정하고 싶은 정보들을 수정한 뒤에 저장 버튼을 누르면 수정한 정보가 반영된 마이페이지가 뜹니다.

4. 그룹 생성
상단의 TRAVELGROUP에서 GROUP CREATE를 클릭하면 그룹을 생성할 수 있는 폼이 나옵니다. 
조건들을 입력하고 생성하기 버튼을 누르면 그룹이 만들어집니다. 입력한 여행 그룹 명이 이미 있는 이름이면 알림창이 뜨고 다시 정보를 입력해야 합니다. (여행 그룹명은 20바이트까지 가능)
그리고 여행 그룹 명, 소개, 여행 장소를 입력하지 않아도 생성되지 않습니다.
 또한 성별과 나이대의 조건이 그룹을 만드는 사용자의 성별이나 나이대 정보와 다르면 그룹이 생성되지 않습니다.

위의 계정으로 로그인을 하셨다면 사용자의 성별은 여성이고 나이대는 20대입니다.
성별을 남성으로 택하거나 나이대를 모순으로 선택하면 그룹 생성이 불가합니다. 테스트 해보시길 바랍니다.

5. 그룹 검색
상단의 TRAVELGROUP에서 GROUP SEARCH를 클릭하면 검색을 할 수 있는 폼이 출력됩니다. 원하는 조건을 설정하고 검색하기를 눌러보세요. 
이 때 아무것도 선택하지 않고 정보를 입력하지 않은 채로 검색하기를 누르면 모든 그룹의 목록이 뜹니다. 또한 조건을 부분 선택해도 검색하기가 가능합니다. 

6. 그룹 보기
상단의 TRAVELGROUP에서 GROUP LIST를 클릭하면 전체 스터디그룹 목록이 출력됩니다. 모집인원이 모두 모인 그룹은 이 화면에 출력되지 않습니다.

6. 그룹 팀원 신청하기
그룹 검색이나 그룹 보기를 통하여 원하는 그룹의 여행그룹 명을 클릭하면 그룹의 상세정보가 나옵니다. 
그리고 여기서 신청 코멘트를 작성하고, '팀원 신청하기' 버튼을 눌러 신청할 수 있습니다. 모집조건과 사용자의 정보가 일치하지 않는다면 신청이 거부됩니다.

위의 계정의 경우 20대이므로 '졸업반 코테 스터디'에 신청해보시기 바랍니다. 나이대 조건에 맞지 않아 가입 신청이 불가한 것을 확인하실 수 있습니다.

(예시) 사용자가 여성이지만 모집성별이 남성인 그룹에 신청할 때, 사용자가 20대이지만 나이대가 10대인 그룹에 신청할 때

5. 나의 신청 현황
상단의 MYHOME에서 MYAPPLICATION을 클릭하면 내가 팀원으로 신청한 그룹의 목록, 수락 여부, 신청날짜를 조회할 수 있습니다. 
수락이 완료된 그룹은 MYGROUP에서 조회할 수 있습니다. 

6. 나의 그룹 보기
상단의 MYHOME에서 MYGROUP을 클릭하면 내가 속해 있는 그룹의 목록이 출력됩니다.
탈퇴를 누르면 곧바로 탈퇴가 진행됩니다. 신중히 클릭하여 주세요. 내가 팀장일 경우 탈퇴가 불가능합니다. 
목록 중 그룹 명을 택하면 그룹 멤버들과 소통할 수 있는 게시판이 출력됩니다.
글쓰기를 누르면 게시글을 작성할 수 있고, 게시글을 누르면 댓글을 작성하거나 삭제할 수 있습니다. 
내가 작성한 게시글을 수정, 삭제 가능하고 댓글도 삭제가 가능합니다.

7. 그룹 관리 
상단의 MANAGE를 클릭하면 사용자가 팀장인 그룹 목록이 출력됩니다. 그룹 명을 클릭하면 그룹의 정보와 멤버 리스트, 신청자 정보를 조회할 수 있습니다. 

신청자 리스트에서 수락/거절 버튼을 눌러 테스트해보세요.
모집인원이 충족된 상태에서 신청자 수락을 누르면 경고창이 뜨면서 신청자 수락이 거부됩니다.

하단의 '정보수정'을 클릭하면 그룹의 정보를 수정하는 폼이 출력됩니다. 
그룹을 생성했을 때 성별과 나이대를 '상관없음'으로 택했다면 다른 성별로 수정할 수 없고, 모집나이대는 자신의 나이대로만 수정 가능합니다. 

성별이 특정성별(여성 혹은 남성)이었다면 그 특정성별의 반대성별로는 수정이 불가능합니다. 
나이대가 20대이었다면 20대나 상관없음 이외로는 변경이 불가능 합니다. 

팀장을 바꾸고 저장하기를 누르면 다시 /manage/main.jsp로 이동하고, 다른 사람으로 팀장을 바꿨지만 그 상태에서 정보 수정을 하면 다시 수정을 할 수 있다.
하지만 팀장을 바꾸고 다른 페이지로 이동했다가 /manage/main.jsp로 이동하면 다른 사람으로 팀장이 바꼈기에 정보 수정이 불가하다.
